# RACI（V1）

## 角色定义
- 麦肯锡顾问：负责“决策问题定义”和“证据取舍”。
- 张小龙：负责“信息架构”和“阅读路径”。
- 前端女王：负责“界面视觉”和“交互落地”。

## 职责矩阵
| 模块 | 设计(Owner) | 评估(Reviewer) | 解决方案(Decision) |
|---|---|---|---|
| 日报框架 | 麦肯锡顾问 | 张小龙 | 麦肯锡顾问 |
| 帖子浏览路径 | 张小龙 | 前端女王 | 张小龙 |
| 可视化UI与交互 | 前端女王 | 张小龙 | 前端女王 |
| 方案对比规则 | 麦肯锡顾问 | 张小龙 | 麦肯锡顾问 |
| 可用性与样式问题修复 | 前端女王 | 麦肯锡顾问 | 前端女王 |

## 执行规则
1. 每次迭代先出“设计稿/Prompt草案”。
2. 由评估角色给出问题清单。
3. 由解决方案角色定稿并落地。
